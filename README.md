# 🧮 Calculadora Simples - Kotlin

Este projeto é uma aplicação desenvolvida em **Kotlin** que realiza operações matemáticas básicas. O objetivo principal foi explorar a sintaxe da linguagem e a lógica de programação voltada para o desenvolvimento Android/Mobile, focando em simplicidade e eficiência.

## 🚀 Funcionalidades

* **Operações Básicas:** Realiza soma, subtração, multiplicação e divisão.
* **Tratamento de Dados:** Lógica para garantir a precisão dos cálculos e evitar erros comuns (como divisão por zero).
* **Interface Limpa:** Foco na usabilidade e na resposta rápida aos comandos do usuário.
* **Código Moderno:** Uso de recursos nativos do Kotlin para um desenvolvimento mais conciso e seguro.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Kotlin
* **Ambiente de Desenvolvimento:** Android Studio / IntelliJ IDEA
* **Conceitos:** Lógica de programação, variáveis, condicionais e funções matemáticas.
